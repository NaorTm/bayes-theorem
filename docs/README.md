# Documentation Index

This directory contains project documentation beyond the root `README.md`.

## Documents

- `ARCHITECTURE.md`: application structure, route map, data flow, and major components
- `CONTENT_GUIDE.md`: how tutorial modules, labs, examples, and practice content are modeled and extended
- `DEVELOPMENT.md`: local workflow, build commands, and troubleshooting
- `AGENTS_ACCEPTANCE.md`: mapping from `AGENTS.md` acceptance criteria to current implementation

## Who Should Read What

- New contributor: start with `DEVELOPMENT.md`, then `ARCHITECTURE.md`
- Content author: start with `CONTENT_GUIDE.md`
- Reviewer or maintainer: use `AGENTS_ACCEPTANCE.md`
